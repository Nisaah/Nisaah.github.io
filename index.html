Security Detection Use Cases

The following detection rules were created to simulate common SOC monitoring use cases.

Detection ID Detection Name Data Source Purpose MITRE ATT&CK
SOC-LAB-001 Multiple Failed Login Attempts Windows Security Event ID 4625 Detect possible brute-force or password guessing activity. T1110 - Brute Force
SOC-LAB-002 Successful Login After Failed Attempts Windows Security Event IDs 4624 and 4625 Identify possible successful login after multiple failed attempts. T1110, T1078
SOC-LAB-003 PowerShell Execution Monitoring Sysmon Event ID 1 Monitor PowerShell execution and review command-line activity. T1059.001 - PowerShell
SOC-LAB-004 Sysmon Process Creation Monitoring Sysmon Event ID 1 Monitor process creation, command line, user, and parent process relationships. T1059 - Command and Scripting Interpreter
SOC-LAB-005 New Windows Service Installed Windows Event ID 7045 Detect new Windows service creation that may indicate persistence. T1543.003 - Windows Service
SOC-LAB-006 User Added to Local Administrators Group Windows Security Event ID 4732 Detect local privilege group modification and possible privilege escalation. T1098 - Account Manipulation
