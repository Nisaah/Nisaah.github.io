```

Built a home-based SOC monitoring lab using Splunk Enterprise to collect, analyze, detect, and investigate security events from Windows Security Logs and Sysmon logs. The project simulates practical SOC activities such as log ingestion, detection engineering, alert validation, dashboard monitoring, and incident documentation.

Project Objectives

  • Build a practical SOC lab using Splunk Enterprise.
  • Collect Windows Security, System, Application, and Sysmon logs.
  • Create SPL-based detection rules for authentication, process, service, and privilege activity.
  • Configure scheduled alerts and validate triggered alerts.
  • Create a SOC monitoring dashboard for investigation visibility.
  • Prepare an incident report based on detected activity.

Lab Architecture

  • Endpoint: Windows laptop
  • SIEM: Splunk Enterprise
  • Forwarding: Splunk Universal Forwarder
  • Log Sources: Windows Security, System, Application, Sysmon
  • Index: soc_lab
  • Detection Language: SPL
