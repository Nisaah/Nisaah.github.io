🛡 Splunk SOC Detection Lab

Built a SOC monitoring lab using Splunk Enterprise to analyze Windows logs, detect brute force attacks, monitor PowerShell activity, and create security dashboards.

⚙️ Key Features

  • Windows Security + Sysmon log ingestion
  • Brute force detection (EventCode 4625)
  • PowerShell execution monitoring
  • Sysmon process tracking
  • Privilege escalation detection

🚨 Security Use Cases

  • T1110 – Brute Force Detection
  • T1059.001 – PowerShell Monitoring
  • T1078 – Valid Accounts

📊 Outputs

  • Dashboards in Splunk
  • Automated alert rules
  • Incident response reports
